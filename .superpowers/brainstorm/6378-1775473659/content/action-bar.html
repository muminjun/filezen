하단 액션바 — 회전 UX 선택

선택된 이미지들을 회전시키는 방식. 어떤 게 더 직관적인가요?

3개 선택됨
직접 입력 °

프리셋 버튼 클릭 시 즉시 회전 적용 → ZIP 저장은 별도 버튼

A — 프리셋 + 직접 입력 분리

90/180/270 버튼은 누르는 즉시 적용. 직접 입력은 숫자 쓰고 "적용" 버튼.

3개 선택됨
회전 °

프리셋 버튼은 입력창에 숫자만 채워줌 → "회전 후 ZIP 저장" 한 번에 실행

B — 입력창 통합 + 원클릭 실행

90/180/270 클릭하면 입력창에 값만 채워줌. "회전 후 ZIP 저장" 버튼 하나로 회전+다운로드 한번에.
