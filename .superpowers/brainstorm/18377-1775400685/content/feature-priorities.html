어떤 걸 만들까요?

관심 있는 항목을 모두 클릭해 주세요 (복수 선택 가능)

🔧 핵심 처리 (현재 미구현)

A

실제 이미지 변환 연결

포맷 변환(JPG/PNG/WebP), 리사이즈, 품질 설정을 실제로 적용. 지금은 설정만 있고 처리가 안 됨.

B

처리 진행률 + 결과 미리보기

변환 중 진행바 표시, 완료 후 원본↔변환 전/후 비교 뷰 연결.

✨ 새 기능

C

이미지 크롭

선택한 이미지를 자유롭게 자르기. 비율 고정(1:1, 16:9 등) 옵션 포함.

D

밝기 / 대비 / 채도 조절

Canvas filter를 이용한 기본 보정. 일괄 적용 가능.

E

일괄 파일명 변경

패턴 입력으로 이름 일괄 변경. 예: photo_{n:001}.jpg

F

워터마크

텍스트 또는 이미지 워터마크를 일괄 삽입. 위치/투명도 조절.

🎨 UI/UX 개선

G

라이트박스 (이미지 크게 보기)

썸네일 더블클릭 시 전체화면 미리보기. 좌우 화살표로 탐색.

H

썸네일 크기 조절 슬라이더

갤러리 썸네일 크기를 작게~크게 실시간 조절. 많은 이미지 탐색 편의성 향상.

I

정렬 + 필터

이름/크기/날짜 정렬. 포맷별 필터(JPG만 보기 등). 검색.

J

키보드 단축키

⌘A 전체선택, Delete 삭제, ←→ 탐색, Space 선택 토글.

K

드래그로 이미지 순서 변경

갤러리에서 드래그 앤 드롭으로 순서 재배치. ZIP 저장 순서에 반영.
