Phase 1 — 설계 개요

승인하면 바로 스펙 문서 작성 → 구현 계획으로 넘어갑니다

A+B — 이미지 변환 연결 + 진행률/비교

변환 흐름 (현재 → 완성 후)
현재 (broken)
선택 → 설정 변경 → 변환하기 클릭 → 아무것도 안 됨
완성 후
선택 → 설정 변경 → 변환하기 클릭
→ 회전 + 포맷변환 + 리사이즈 동시 적용
→ 툴바에 진행률 표시 (n/총n장)
→ 완료 시 PreviewPanel에 전/후 비교
→ ZIP 다운로드
변경 파일:
  • AppContext.tsx — processFiles() 실제 구현 (imageProcessor 호출)
  • RotationToolbar.tsx — 변환하기 버튼을 processFiles + ZIP으로 연결
  • PreviewPanel.tsx — processedUrl 연결, 전/후 비교 뷰 활성화

G — 라이트박스

동작 방식
썸네일 더블클릭 → 전체화면 오버레이 열림
← → 화살표로 이미지 탐색 / ESC 닫기 / 처리된 이미지면 전/후 탭 전환
새 파일: src/components/upload/LightboxModal.tsx

H — 썸네일 크기 슬라이더

갤러리 상단 컨트롤바
갤러리 위에 한 줄 컨트롤바 추가:
[전체선택] [n장 선택됨] ———————— [🔲 ——슬라이더—— 🔳]
슬라이더 값 → CSS grid grid-template-columns 조절 (2~8열)
변경: ImageGallery.tsx 상단에 컨트롤바 추가
